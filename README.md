## Technologies
Project is created with:

* Flask 1.1.2
* Werkzeug 1.0.1
	
## Setup
To run this project, install it locally using the command prompt, terminal, etc.:

```
$ https://github.com/jkiefn1/Text-Similarity.git
$ cd Text-Similarity
$ $ python3 api.py
```
